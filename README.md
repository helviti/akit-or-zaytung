# akit-or-zaytung

Şakayla gerçeği ayırt etmek isteyenler için Sveltekit ve Tailwind bazlı bir oyun.

## Kurulum

```bash
npm install

# development için
npm run dev

# buildlemek için
npm run build

# buildlediğiniz dosyaları önizlemek için
npm run preview

# datanın alındığı news.json'u yaratmak için
node src/utils/news_fetch_utility.js

```
